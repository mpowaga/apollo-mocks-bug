All results has been generated by running following query:

```
{
  posts {
    title
    authors {
      name
    }
  }
}
```

# good

```javascript
const typeDefinitions = `
type User {
  name: String
}

type Post {
  title: String
  authors: [User]
}

type Query {
  posts: [Post]
}

schema {
  query: Query
}
`;
```
